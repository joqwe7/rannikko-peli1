# Pelimekaniikat ja Vuorovaikutukset

## F. Pelimekaniikat

### Pelaajien toiminnot

#### 1. Rakentaminen
- Kustannus: 50€
- Cooldown: 30s
- Rakennustyypit:
  * Aallonmurtaja (suojaa eroosiolta)
  * Tutkimuskeskus (+2 TP/min)
  * Ympäristöasema (+1 YP/min)

#### 2. Tutkimus
- Kustannus: 30 TP
- Cooldown: 45s
- Tutkimusalueet:
  * Eroosiomallit (+10% puolustus)
  * Biodiversiteetti (+5 YP/min)
  * Infrastruktuuriteknologia (-20% rakennuskustannukset)

#### 3. Investointi
- Kustannus: 100€
- Cooldown: 60s
- Kohteet:
  * Kestävä kalastus (+3€/min)
  * Ekoturismi (+2€/min, +1 YP/min)
  * Tuulivoima (+5€/min, kertaluontoinen -50 YP)

#### 4. Hyökkäys/Sabotaasi
- Kustannus: 75€ + 20 TP
- Cooldown: 90s
- Tyypit: (katso PvP-osio)

#### 5. Puolustus
- Kustannus: 50% hyökkäyksen hinnasta
- Cooldown: 20s
- Aktivoituu automaattisesti hyökkäyksen aikana

#### 6. Tiedustelu
- Kustannus: 25 TP
- Cooldown: 30s
- Näyttää kohdetiimin resurssit ja rakennukset

### MCQ/Minipeli aktivointi
- Jokainen toiminto vaatii minipelin/kysymyksen
- Onnistuminen: Toiminto toteutuu
- Epäonnistuminen: Menettää 50% resursseista, toiminto peruuntuu

### Passiivinen tulonmuodostus
- Perustulo: 5€/min
- Perustutkimus: 2 TP/min
- Ympäristön palautuminen: 1 YP/min
- Rakennukset lisäävät tuloja
- Max kapasiteetti: 1000€, 500 TP, 300 YP

### Griefing-esto ja reiluus
- Max 3 hyökkäystä/tiimi/peli
- Min. 60s väli hyökkäysten välillä
- Automaattinen tasoitus jos ero >300 pistettä
- Resurssihyvitys jos pelaaja poistuu kesken
